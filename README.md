# 🎙️ LingoCast

**LingoCast** 是一款開源、極低延遲的 **「即時雙語對照字幕與大會投屏系統」**。

它能將您的口述語音秒級轉化為網頁端的雙語對照字幕。專為大會演講、實體論壇投屏、跨國商務會議、教學演示等需要即時投影字幕的場合而設計。

---

## ✨ 核心功能

- **🎙️ 網頁端免驅動錄音**：投影用的電腦無需安裝任何特殊的音效驅動或進行繁瑣的麥克風設定。只需打開瀏覽器，授權麥克風存取，即可立即開始發音並投影。
- **⚡ 雙引擎極速同譯管道**：
  - **語音轉文字 (STT)**：基於 Groq 雲端託管的 `whisper-large-v3-turbo`，實現毫秒級語音轉譯。
  - **雙語翻譯 (LLM)**：基於 `llama-3.3-70b-versatile`，生成符合語境、中英上下排版對照的流暢大字幕。
- **📖 自訂專用語字典安全護欄**：
  - **名詞替換對齊**：可強制將特定詞彙替換為指定術語（如 `Kavira` ➔ `n7 Kavira`、`Agent` ➔ `AI 代理人`）。
  - **單獨名詞原樣保留**：在字典中若只填寫「輸入詞」而將「取代為」留空，系統會將其視為專有名詞，在轉錄與翻譯中**強制原樣保留**，不作任何詞義變更或翻譯。
  - **網頁端字典編輯器**：可在網頁上直接開啟彈窗編輯自訂詞彙，點擊儲存自動同步至後端。
- **🎨 專業大會投影 UI**：
  - 採用符合高級設計美學的暗黑科技風格 (Sleek Dark Mode) 與藍紫漸變效果。
  - 超大字體、高對比度的雙語字幕展示區，專為投影機和大會螢幕優化。
  - SVG 微型聲波動畫，能根據錄音狀態進行起伏反饋。
  - **一鍵清除畫面 🧹**：點擊即可清空目前螢幕上的所有歷史字幕。
- **🛡️ 100% 繁體中文保證**：後端全面整合 `OpenCC` 繁簡轉換引擎，保證所有輸出的中文原文與譯文皆為標準繁體中文，絕無簡體字污染。

## 🛠️ 安裝與部署指南

### 環境要求

- Python 3.10 或更高版本。
- 一組免費的 Groq API 金鑰（可至 [console.groq.com](https://console.groq.com/) 申請）。

### 1. 下載專案

```bash
git clone https://github.com/您的用戶名/LingoCast.git
cd LingoCast
```

### 2. 安裝 Python 依賴包

```bash
pip install -r requirements.txt
```

### 3. 配置環境變數

將 `.env.example` 複製並命名為 `.env`：

```bash
cp .env.example .env
```

打開 `.env` 檔案並填入您的 Groq API 金鑰：

```env
GROQ_API_KEY=gsk_您的金鑰_寫在這裡
```

## 🚀 啟動 LingoCast

### 方案 A：Web 網頁版 (推薦大會投屏使用)

1. 在終端機啟動 FastAPI 後端服務：
   ```bash
   python app.py
   ```
2. 打開瀏覽器訪問以下網址：
   * **本機電腦**：[http://localhost:8090](http://localhost:8090)
   * **跨設備連線（如 iPad、手機、投影主機）**：`http://<您的 Mac 區域網 IP>:8090`（需在同一區域網內）。
3. 允許麥克風權限，選擇「手動對講」或「連續投屏」模式，即可開始投影！

### 方案 B：CLI 終端機版

若您只想在 Command Line 介面中測試翻譯，可以執行：

* **互動翻譯模式**：
  ```bash
  python cli.py
  ```
* **功能演示模式**（展示內建翻譯對齊效果）：
  ```bash
  python cli.py --demo
  ```

## ⚙️ 管理自訂字典

您可以透過兩種方式管理字典：
1. **網頁 UI 編輯**：點選網頁右上角的 **「⚙️ 自訂字典」** 按鈕，即可彈出編輯視窗。
2. **直接修改設定檔**：在專案根目錄下直接編輯 `dict.json` 檔案：
   ```json
   {
     "Kavira": "n7 Kavira",     // 將 "Kavira" 強制替換為 "n7 Kavira"
     "Antigravity": "",          // 值留空，將強制 "Antigravity" 在翻譯中保持原樣不變
     "Systems Thinking": "系統思考"
   }
   ```
